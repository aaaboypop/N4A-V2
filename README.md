# N4A-V2
N4A Video Tools

This GUI Tools for video/picture

### Feature
 - Extension support
   - Video (avi mp4 mkv wmv ts mov)
   - Audio (aac ogg mp3 pcm)
   - Image (jpg png bmp)
 - ffmpeg features
   - Decoding Test
   - Convert files video > image, image > video
   - Mux & Demux Video & Audio
   - Remove Duplicate Frame
   - Create Thumbnail
   - Resize Video
   - Deinterlace
   - Trim
 - Image Upscale with Waifu2X (CUDA & Vulkan)
    - Support Multi GPU
